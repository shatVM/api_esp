# Підключення модулів до ESP8266

Цей документ описує, як підключати зовнішні модулі та сенсори до плати ESP8266 для цього проєкту.

## Шина I2C (Сенсори AHTx0 та BH1750)

Сенсори температури/вологості (AHTx0) та освітленості (BH1750) працюють по шині I2C.

| Пін ESP8266 | Назва на платі | Пін на модулі |
| :---------- | :------------- | :------------ |
| `GPIO4`     | `D2`           | `SDA`         |
| `GPIO5`     | `D1`           | `SCL`         |

*   **Живлення:** Не забудьте підключити `VCC` (зазвичай 3.3V) та `GND` для обох модулів.

## Сенсор DHT11

Цифровий сенсор температури та вологості DHT11.

| Пін ESP8266 | Назва на платі | Пін на модулі |
| :---------- | :------------- | :------------ |
| `GPIO2`     | `D4`           | `DATA` / `OUT`|

*   **Живлення:** Підключіть `VCC` та `GND`.
*   **Підтягуючий резистор:** Для стабільної роботи може знадобитися підтягуючий резистор (pull-up) номіналом 4.7kΩ - 10kΩ між пінами `DATA` та `VCC`.

## Аналоговий вхід (Вимірювання напруги)

Для моніторингу напруги використовується аналоговий пін `A0`.

| Пін ESP8266 | Призначення                               |
| :---------- | :---------------------------------------- |
| `A0`        | Вхід для вимірювання аналогового сигналу. |

*   **Увага:** Максимальна вхідна напруга на піні `A0` для ESP8266 становить **1.0V**. Для вимірювання більшої напруги (наприклад, від батареї) необхідно використовувати дільник напруги. У коді заданий коефіцієнт `VOLTAGE_DIVIDER_RATIO = 5.0`, що передбачає наявність дільника.

## Керовані виходи

Ці піни налаштовані як виходи для керування зовнішніми пристроями (наприклад, реле, світлодіодами).

| Пін ESP8266 | Назва на платі | Призначення |
| :---------- | :------------- | :---------- |
| `GPIO12`    | `D6`           | Вихід 1     |
| `GPIO13`    | `D7`           | Вихід 2     |
| `GPIO14`    | `D5`           | Вихід 3     |
